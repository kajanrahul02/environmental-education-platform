# 🌱 Gamified Environmental Education Platform

## Project Overview

The Gamified Environmental Education Platform is a web-based mini project designed to increase environmental awareness among students through interactive learning and gamification. The platform provides educational content on environmental topics and allows students to test their knowledge through quizzes while earning points, badges, and rewards.

This project helps students learn important concepts such as Water Conservation, Renewable Energy, Waste Management, Tree Plantation, and Pollution Control in an engaging and enjoyable way.

* Student Login System

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in a web browser.
4. Enter your name and roll number.
5. Read the learning materials.
6. Complete the environmental quiz.
7. View your score, badge, and leaderboard.
